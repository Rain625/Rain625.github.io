# Hanxiaoyu 个人主页

这是一个简洁的学术个人主页模板。

## 文件结构

```
├── index.html    # 主页面
├── style.css     # 样式文件
├── script.js     # 交互脚本
└── README.md     # 说明文档
```

## 功能特性

- ✅ 响应式导航栏，包含以下栏目：
  - About（关于）
  - Publications（出版物）
  - Repositories（代码仓库）
  - Talks（演讲）
  - Teaching（教学）
- ✅ 平滑滚动效果
- ✅ 移动端适配
- ✅ 简洁现代的设计风格

## 使用方法

1. 直接在浏览器中打开 `index.html` 即可查看效果
2. 后续可以根据需要填充各个栏目的内容
